## The Problem

Each team built UI differently.

Buttons looked different. Forms behaved differently.

Consistency didn’t exist.

## The Approach

We started small:

- Button
- Input
- Modal

Then scaled.

## The Rule

Every component had to be:

- Accessible
- Reusable
- Documented
